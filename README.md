# Why use?
This docker-compose yml will help us to create local environment for PostgresSQL and AWS

# How to use
* `docker-compose up` to create environment
* `docker-compose down` to down environment

## PostgresSQL configurations
* host : local host
* port: 5432
* database: postgres
* username: postgres
* password: postgres
